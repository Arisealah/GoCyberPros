Supported Extractors & Features

TXT Extractor

Quickly parse plain text files to identify key entities, sentiments, and structured information.

  • Supports .txt, .log, .csv (text-based)
  • Extracts names, dates, locations, organizations
  • Sentiment analysis for text blocks

DOCX Extractor

Advanced parsing for Microsoft Word documents to retrieve content, comments, and hidden properties.

  • Extracts document properties (creator, company)
  • Content and revision history extraction
  • Supports .docx and .doc (via conversion)

CSV Extractor

Extracts structure, headers, and statistics from CSV files.

  • Column and row analysis
  • Header detection
  • Data type inference

EML Extractor

Extracts metadata and content from email files.

  • Sender, recipient, subject, date
  • Attachment info
  • Body and headers

JPG/PNG Extractor

Extracts image metadata, EXIF, and technical details.

  • Dimensions, color mode
  • EXIF and GPS data
  • File info

JSON Extractor

Analyzes JSON structure and provides statistics.

  • Key and value analysis
  • Depth and type info
  • File info

MP3/MP4 Extractor

Extracts audio/video metadata and technical details.

  • Duration, bitrate, codec
  • Tags and properties

PDF Extractor

Extracts metadata, text, and structure from PDF files.

  • Author, creation date
  • Page count, content summary

PPTX Extractor

Extracts metadata and slide info from PowerPoint files.

  • Slide count, titles
  • Author, creation date

XLSX Extractor

Extracts metadata and sheet info from Excel files.

  • Sheet names, cell stats
  • Author, creation date

XML Extractor

Analyzes XML structure and provides statistics.

  • Element and attribute analysis
  • Depth and structure

ZIP Extractor

Extracts archive contents and compression info.

  • File list and sizes
  • Compression methods
